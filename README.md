# Banking Management System

A console-based Banking Management System built using Java, JDBC, and MySQL.

## Features
- Create account
- Deposit money
- Withdraw money
- Check balance
- View transaction history

## Tech Stack
- Java
- JDBC
- MySQL
- OOP

## How to Run
1. Clone repository
2. Open in VS Code
3. Set up MySQL database
4. Run schema.sql
5. Update database credentials in BankingApp.java before running.
6. Run Main.java
